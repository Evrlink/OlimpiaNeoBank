# Mobile App

React Native **Olimpia** app for **iOS and Android**.

## What belongs here

- React Native application source (screens, navigation, native config)
- Privy auth integration (mobile SDK)
- App entry points, tab/stack navigators, and platform-specific assets
- iOS and Android project configuration (Xcode, Gradle)

## Out of scope

- Marketing site (see `apps/marketing/`)
- Shared visual tokens and components (see `packages/design-system/` and `packages/ui/`)
- Backend orchestration — see `apps/api/`

## Status

Folder scaffold only — no dependencies or app code yet.
