# UI Components

Reusable UI components shared across **Olimpia** surfaces.

## What belongs here

- Presentational React / React Native components (buttons, inputs, cards, lists, modals)
- Layout primitives and composition patterns used by marketing and mobile
- Component APIs documented for consistent usage

## Out of scope

- Raw design tokens (colors, typography, spacing) — see `packages/design-system/`
- App-specific screens and navigation — see `apps/mobile/` and `apps/marketing/`
- Business logic and API calls

## Status

Folder scaffold only — no dependencies or components yet.
